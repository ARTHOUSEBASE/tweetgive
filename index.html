How to Give with @bankrbot

The easiest way to donate on Base. Just tweet.

1

Tweet @bankrbot

Recommended

Send a personal tweet mentioning @bankrbot with hashtag #tweetgiveonbase

Format:

"Hey @[creator], [personal message]. Sending [amount] ETH @bankrbot #tweetgiveonbase [campaign-id]"

Example:

"Hey @sekolahkode, incredible work with the coding school in Papua! Sending 0.5 ETH to support the mission 🎓 @bankrbot #tweetgiveonbase 123"

  1. Write a personal tweet with your message
  2. Include @bankrbot mention
  3. Add #tweetgiveonbase hashtag
  4. Specify amount and campaign ID
  5. @bankrbot will reply with payment link
  6. Click link → confirm in wallet → done!
